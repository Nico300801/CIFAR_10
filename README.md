# Clasificación de imágenes con CIFAR-10 usando CNN

Este repositorio contiene un proyecto de **clasificación de imágenes** utilizando el dataset **CIFAR-10** y una **Red Neuronal Convolucional (CNN)** implementada en Python mediante Jupyter Notebook.

## Contenido del repositorio

- **`CNN_CIFAR10.ipynb`**  
  Notebook principal donde se carga el dataset, se define la arquitectura de la CNN, se entrena el modelo y se evalúan los resultados.

- **`cifar10_mini.npz`**  
  Versión reducida del dataset CIFAR-10 en formato NumPy, utilizada para entrenar y probar el modelo de forma más rápida.

- **`Garcia_Nicolas_cifar10.pdf`**  
  Documento explicativo del proyecto, donde se describen los objetivos, la metodología y los resultados obtenidos.

## Dataset CIFAR-10

CIFAR-10 es un conjunto de datos de clasificación de imágenes que contiene **60.000 imágenes en color (32x32 píxeles)** distribuidas en **10 clases**:

- avión
- automóvil
- pájaro
- gato
- ciervo
- perro
- rana
- caballo
- barco
- camión

## ⚙️ Requisitos

Para ejecutar el notebook se recomienda tener instalado:

- Python 3.x
- NumPy
- Matplotlib
- TensorFlow y/o Keras
- Jupyter Notebook
